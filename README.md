# Լաբորատոր աշխատանք - RLE սեղմում

[cite_start]Այս ծրագիրը իրականացնում է տվյալների սեղմում Run-length encoding (RLE)՝ հերթականնությունների երկարությունների կոդավորում [cite: 7] ալգորիթմով։ Գրված է Python-ով։

## Ինչպես օգտագործել

1. Ստեղծեք `input.txt` ֆայլ ճիշտ **նույն պապկայի մեջ**, որտեղ գտնվում է Python կոդը։
2. [cite_start]Մեջը գրեք տեքստ, որտեղ կան կրկնվող տառեր, օրինակ՝ `AAAAAAAABCCCC`[cite: 94]:
3. Աշխատացրեք սկրիպտը։
4. Տերմինալում ընտրեք `1` (սեղմելու համար) կամ `2` (ապակոդավորելու/հետ բերելու համար)։
5. Ծրագիրը կվերագրանցի ֆայլը նոր արդյունքով։

## Կարևոր նշում դասախոսի համար

Այս բազային տարբերակը լավ աշխատում է միայն այն ժամանակ, երբ կան շատ կրկնվող սիմվոլներ։ Եթե փորձենք սեղմել սովորական տեքստ (որտեղ տառերը չեն կրկնվում), ֆայլի չափը կմեծանա, քանի որ ամեն տառի դիմաց ավելանում է նրա քանակը (1 հատ պրոբելը դառնում է `1 `)։ 
[cite_start]Որպեսզի ալգորիթմը ճիշտ աշխատի ցանկացած տեքստի հետ, պետք է ավելացնել հատուկ բայթ [cite: 115][cite_start], որի առաջին բիտը ցույց է տալիս կրկնվում է , թե ոչ [cite: 116][cite_start], և առանձնացնել չկրկնվող հաջորդականությունը [cite: 118] կրկնվողից։ Այս սկրիպտը գրված է ալգորիթմի բազային տրամաբանությունը ցույց տալու համար։
